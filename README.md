# dataset-DZ3
https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset?resource=download
В датасете набор данных о розничных продажах и характеристиках клиентов! Этот набор данных был тщательно разработан для имитации динамичной покупательской способности. 
Предоставляя идеальную площадку для анализа на розничных продажах и характеристиках клиентов, для извлечения сложных закономерностей, подготовки выводов и понятия поведение клиентов.
Он включает в себя ключевые данные, такие как идентификатор транзакции, дата, идентификатор клиента, пол, возраст, категория товара, количество, цена за единицу и общая сумма.
Эти атрибуты позволяют проводить многостороннее исследование тенденций продаж и покупательского поведения.
